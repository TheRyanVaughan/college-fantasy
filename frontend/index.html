Teams In League


Basic Functionality

Get League

Returns all the teams in a given league

Get Teams for User

Returns all the teams for a given user

Get Team

Returns all the players on a given roster

Get Player

Returns information about a specified player if given a first and last name

Add Team

Adds a team to a specified league


Drop Team

Drops a team to a specified league

Add League

Adds a league to the database


Drop League

Drops a league from the database

Advanced Functionality

Calculate Score

Create a mathematical equation to apply to PPG, RPG, APG, etc. to give a team score based on team performance.

Drop Player

Gets rid of player from a team roster and adds them to Free Agents

Add Player

Adds a player from Free Agents and removes them from the free agency list

Trade Player

Given two teams and a player from each, have them switch rosters
